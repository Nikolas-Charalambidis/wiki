== Versions and Environments

:label: Java versions
:date: 17-08-2022
include::partial$item.adoc[]

[,shell]
----
# Java home
/usr/libexec/java_home

# Java certain version
/usr/libexec/java_home -v 17

# Java all versions
/usr/libexec/java_home -V
----
